# Requirements — Leafy Flavours Web Menu

## Must Have (Phase 1 — Launch)

- [x] Single-page website with all 4 experiences described
- [x] Hero section with full-screen photo and tagline
- [x] About / Why Us section with elevated pitch
- [x] Experience sections with photos and culturally resonant copy
- [x] Gallery section with real event photos
- [x] Theme matching section (Pastel Baby Shower, Bollywood Night, etc.)
- [x] Contact section with WhatsApp button + call button
- [x] PDF download button
- [x] Mobile-first responsive design
- [x] Brand colours (Teal + Gold) and Google Fonts
- [ ] Live on GitHub Pages with shareable URL
- [ ] Git repository initialised and pushed

## Should Have (Phase 2 — Polish)

- [ ] Smooth scroll navigation between sections
- [ ] Experience pill navigation (sticky sub-nav)
- [ ] Fade-in animations on scroll
- [ ] WhatsApp pre-filled message ("Hi Deepika, I'm interested in...")
- [ ] Page title and meta description for sharing previews

## Nice to Have (Phase 3 — Growth)

- [ ] Custom domain (leafyflavours.com.au or similar)
- [ ] Instagram embed or feed link
- [ ] Enquiry form (Formspree — free, no backend needed)
- [ ] Google Analytics (basic visitor tracking)
- [ ] Separate pages per experience (if Deepika wants deeper SEO)

## Out of Scope

- Backend / database / CMS
- Online ordering or payment
- Booking calendar
- User accounts
