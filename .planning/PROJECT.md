# Leafy Flavours — Web Menu

## What This Is

A shareable website Deepika sends to clients via WhatsApp instead of a PDF.
Hosted on GitHub Pages — free, permanent URL, mobile-friendly.
Single static HTML file. No frameworks. No build step. Edit anywhere.

## Business Context

**Leafy Flavours by KitchenChimes**
Owner: Deepika Nagaraja | Kellyville, Sydney | @leafyflavours
Boutique Karnataka food experience service — styled, served, packed at client's venue.
Cuisine: Mysuru/Karnataka + Indian street food/chaats.
Target: Indian community in Sydney for birthdays, baby showers, housewarmings.

**The problem this solves:** Clients used to get a PDF that looked like a takeaway order form.
They compared individual item prices to bulk caterers → price shock → lost orders (5 in 1 month).
Solution: Named experience packages, no prices shown, contact for quote.

## The Four Offerings

1. **THE STREET TABLE** — Indian street food as event centrepiece
2. **THE KARNATAKA TABLE** — Three tiers (Tindi Habbada / Namma Habba / Rajara Bendige)
3. **ELLE OOTA HERITAGE STYLE** — Traditional 6-course Karnataka sit-down, 8–20 guests
4. **THE OPENING ACT** — Starters + styling only, client brings their own mains

## Users

**Primary:** Deepika — edits content, adds photos, tweaks copy
**End users:** Sydney Indian community — browses on mobile via WhatsApp link

## Success Criteria

- Link is live and shareable from GitHub Pages
- Works on mobile (WhatsApp in-app browser)
- Deepika can edit content from her phone in < 5 minutes
- No prices visible on site
- Download PDF button works
- WhatsApp and call buttons work

## Constraints

- No frameworks, no build tools, no npm — plain HTML/CSS only
- GitHub Pages free tier (no custom domain needed unless Deepika wants one later)
- HEIC photos must be pre-converted to JPG before adding to `images/`
